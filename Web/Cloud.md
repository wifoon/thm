**IaaS (Infrastructure as a Service)** Rent IT infrastructure like servers, virtual machines, and storage from a vendor. Users manage operating systems, apps, and data.
`AWS, Microsoft Azure, Google Cloud Platform`

**PaaS (Platform as a Service)** The cloud provider manages the infrastructure and the operating system. Users focus on building, deploying, and running your application without maintaining the underlying server infrastructure.
`Google App Engine, AWS Elastic Beanstalk, Microsoft Azure App Service`

**SaaS (Software as a Service)** Delivers fully managed, ready-to-use software applications over the internet, usually via subscription.
