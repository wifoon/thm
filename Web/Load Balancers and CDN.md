
**Load Balancers**

Load balancer forward requests to multiple servers instead of one using different algorithms, ensuring high traffic websites can handle the load and providing a failover if a server becomes unresponsive.

Load balancers also perform periodic checks with each server to ensure they are running correctly; this is called a **health check**. If a server doesn't respond appropriately or doesn't respond, the load balancer will stop sending traffic until it responds appropriately again.


**CDN (Content Delivery Networks)**

CDN is a distributed group of servers close to end users, that stores content for websites and applications, reducing latency and accelerating load times. By directing user requests to the nearest **edge server** rather than a central origin server, CDNs improve website performance, security, and reliability.
